# 🚀 RUStress: Бесплатная расстановка ударений онлайн

Интеллектуальный сервис для автоматической расстановки ударений в русском языке с учетом контекста (омографов). Теперь в версии **Truly Portable**.

---

## 🔥 Ключевые особенности
- 🧠 **Context Awareness**: Правильно различает *за́мок* и *замо́к* благодаря нейросетевым моделям.
- 🚀 **100% Portability**: Работает как одиночный `.exe`, не требуя установки Python или библиотек.
- 🛠 **Monkey-patching Engine**: Внедрена система патчинга библиотек в оперативной памяти — больше никакого ручного редактирования системных файлов.
- ⚙️ **Easy Config**: Все исключения и правила (например, для слов *узнаете*, *лет*) вынесены в отдельный файл `config.py`.
- 🔒 **Privacy First**: Текст обрабатывается локально на вашем ПК.

## 🛠 Технологический стек
- **Backend**: Python 3.12 + FastAPI + Uvicorn
- **AI/ML**: RUAccent + ONNX Runtime (с кастомным патчингом в памяти)
- **Frontend**: Чистый HTML/JS (через статические файлы)

## 📦 Как запустить
### Вариант 1: Portable (.exe)
1. Скачайте свежий `RUStress.exe` из раздела **Releases**.
2. Запустите файл.
3. Откройте в браузере: `http://localhost:8001` (сервис сам откроет вкладку).

### Вариант 2: Из исходников (для разработчиков)
```bash
git clone https://github.com/rickinort/-ru-stress-marks.git
pip install -r requirements.txt
python main.py
```

## 🏗 Сборка проекта
Для создания собственного портативного билда используйте:
```bash
pyinstaller --noconfirm RUStress.spec
```
