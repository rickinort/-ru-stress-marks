## Русскиий

**Бесплатная расстановка ударений онлайн** — это интеллектуальный сервис для автоматической расстановки ударений в русском языке с учетом контекста (омографов).

### Почему это круто?
- ✅ **Без лимитов**: Нет ограничений на количество символов. Обрабатывайте целые книги!
- ✅ **Бесплатно**: Не требует подписок и регистраций.
- ✅ **Приватность**: Работает за счет ресурсов вашего ПК, текст не хранится на сторонних серверах.
- 🧠 **Нейросетевые модели**: Использует современные модели для точной расстановки.
- 📖 **Учет контекста**: Правильно различает слова-омографы (например, *за́мок* и *замо́к*).
- 🚀 **Портативность**: Работает как одиночный `.exe` файл, не требует установки.

### Как запустить (Portable)
1. Скачайте `RUStress.exe`.
2. Запустите файл.
3. Откройте `http://localhost:8000` в браузере.
